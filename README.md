## 📖 Sobre o Projeto

Interface de e-commerce desenvolvida em React, TypeScript e Tailwind CSS. Focada em conversão, responsividade e componentização, o projeto exibe produtos high-end em um layout moderno, combinando técnicas avançadas de estilização com uma arquitetura de código profissional.

### ✨ Destaques

- ⚛️ **React + TypeScript** - Stack moderna e altamente tipada
- 🎨 **Tailwind CSS** - Estilização avançada utility-first
- 📱 **Mobile First** - Design totalmente adaptável para qualquer dispositivo
- 🖼️ **Glassmorphism** - Efeitos de luz e background decorativos dinâmicos
- 🔄 **Carrossel Nativo** - Navegação fluida com CSS `scroll-snap` e hook `useRef`
- 🧩 **Componentização (SOLID)** - Estrutura separando lógicas de navegação e design
